How to run this project

1. Copy link git vào thư mục muốn lưu gõ lệnh
- git clone https://gitlab.duthu.net/S52000761/finalproject.git

2. Mở command line (nếu đã download MongooseDB)
- mongod

3. Di chuyển và mở command line (đã cài đặt NodeJS) trong thư mục vừa clone gõ lệnh dưới để tải thư viện cần thiết về cho project hoạt động
- cd finalproject
- npm i

4. Tiếp tục gõ lệnh dưới để khởi động hệ thống
- node app.js

5. Vào link và bắt đầu trả nghiệm
- http://localhost:3000/

Lưu ý: Dữ liệu của phim đã được thêm vào database khi bắt đầu khởi động, tài khoản admin mặc định được lưu trong database là:
- username: "admin"
- password: "123456"
